Приложение для управления Steam-аккаунтами,мобильным аунтификатором и инвентарями.

## Стек

- Backend: Node.js + Express
- Frontend: React
- WebSocket

---

## Установка и запуск

```bash
git clone https://github.com/your-username/steam-manager.git
cd steam-manager

cd backend
npm install
npm start

cd frontend
npm install
npm start
